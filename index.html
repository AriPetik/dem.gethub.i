Из каких имен и на каких уровнях иерархии состоит Доменное имя моего сайта? 
Корневой уровень (Root domain) — обозначается точкой . (обычно не отображается явно).
Домен верхнего уровня (Top-Level Domain, TLD) — .io
Это национальный домен верхнего уровня, изначально присвоенный Британским территориям в Индийском океане, но сейчас широко используется для технологических и развлекательных проектов.
Второй уровень (Second-Level Domain) — github
В данном случае это часть домена, зарегистрированная сервисом GitHub Pages для хостинга пользовательских сайтов.
Третий уровень (Third-Level Domain / поддомен) — aripetik
Это имя пользователя или организации на GitHub, чей репозиторий использует GitHub Pages.
